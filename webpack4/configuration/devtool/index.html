# devtool

webpack 使用该配置项来决定 sourcemap 的生成。在这之前，你必须对 sourcemap 原理有深入的理解。

先用一个简单的 demo 测试一下 webpack 的所有 devtool 配置项。demo 如下：

index.js
import A from './a'
console.log(A)
a.js
export default 'a'
webpack.config.js
module.exports = {
  context: __dirname,
  entry: './index.js',
  optimization: {
    minimize: false
  },
  devtool: '填入下面对应的字符串',
  output: {
    filename: "[name].js"
  }
};

# false

运行打包命令之后，不会生成 map 文件，在生产环境都是这么设置的。
